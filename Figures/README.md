# How to plot

Before to run the user need to edit the fist 3 lines in example_plots.R to add: 
  - dir: Dimulation directory
  - N: pupularion size 
  - M: number of simulations in the folder

The script example_plots.R generate three figures:
  - rank_degree.png 
  - SIRD.png 
  - herd_immunity.png
